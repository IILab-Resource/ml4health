# 案例教程 11：校准分析与决策曲线分析 (DCA)

## 一、前序知识要求

### 1.1 核心概念

| 概念                    | 说明                          | 重要性                   |
| --------------------- | --------------------------- | --------------------- |
| **预测概率**              | 模型的原始输出（0-1 之间的连续值），而非离散的分类 | 两个模型的 AUC 相同但概率分布完全不同 |
| **校准度 (Calibration)** | 预测概率与实际观测比例的一致性             | 决定医生是否信任模型的概率输出       |
| **AUC**               | 排序能力——正类的概率是否系统性地高于负类       | 只关心"顺序"不关心"数值"        |
| **Brier Score**       | 预测概率与真实标签的均方误差              | 同时惩罚排序错误和校准错误         |
| **净获益 (Net Benefit)** | 权衡"正确干预"和"错误干预"的临床获益        | 将统计指标"翻译"为临床决策        |

### 1.2 三个核心问题

本教程围绕三个问题展开：

| 问题               | 回答工具                       | 教学要点         |
| ---------------- | -------------------------- | ------------ |
| 1. 模型预测的概率有多可信？  | 校准曲线 + Brier Score + HL 检验 | 高 AUC ≠ 高校准  |
| 2. 这个模型有临床价值吗？   | DCA (决策曲线分析)               | 统计显著 ≠ 临床获益  |
| 3. 在哪些风险阈值下应该干预？ | DCA 获益范围                   | 不同阈值对应不同临床决策 |

### 1.3 前序教程回顾

- **案例 10（不平衡数据）**：不平衡数据中的概率往往被严重压缩——模型对多数类输出接近 0 的概率

## 三、各模块详解

### 第一部分：校准分析

#### 1.1 校准曲线

校准曲线的原理很简单：将测试集按预测概率排序，分成 10 个等频 bin，在每个 bin 中计算"预测概率的平均值"和"实际正类比例"，然后画点连线。

```
                    ┌──────────────────────┐
                    │   完 美 校 准 曲 线    │
                    └──────────────────────┘
                                │
        预测 0.8 → 实际 80% 是正类    ← 点在对角线上
        预测 0.5 → 实际 50% 是正类
        预测 0.2 → 实际 20% 是正类

                    ┌──────────────────────┐
                    │   校 准 不 良 曲 线    │
                    └──────────────────────┘
                                │
        预测 0.8 → 实际只有 40% 是正类  ← 点在对角线下方
        预测 0.2 → 实际却有 60% 是正类  ← 模型过于自信
```

#### 实验结果

| 模型                  | Brier      | AUC        | 校准曲线位置   | 校准度    |
| ------------------- | ---------- | ---------- | -------- | ------ |
| XGBoost             | **0.1153** | **0.9168** | 接近对角线    | **最佳** |
| Random Forest       | 0.1164     | 0.9162     | 偏低（过度自信） | 良好     |
| KNN (k=15)          | 0.1216     | 0.9048     | 几乎完美匹配   | 中      |
| Logistic Regression | **0.1307** | 0.8944     | 系统性偏差    | **最差** |

#### 1.2 关键教学发现：KNN 的校准度为什么最好但 Brier 不是最低？

这是一个**非常重要的教学陷阱**：

```
KNN 的校准度（Murphy 分解中的 Calibration 分量）最好，
但整体 Brier 并不是最低:

KNN 预测概率的特点:
- 数据轻度不平衡: VIVO 41.15%, MORTO 58.85%
- KNN 的校准分量 = 0.0010 (所有模型中最低)
  → 预测概率与实际比例最接近
- 但 KNN 的 Brier = 0.1216 (不是最低, XGBoost=0.1153 更低)

为什么校准好但 Brier 不是最低?
  Brier = 校准度 - 鉴别力 + 不确定性
        = 0.0010 - 0.1216 + 0.2422 = 0.1216
  → KNN 的鉴别力 (Refinement) 较高, 部分抵消了校准优势

但 Recall = 0.8335 → 模型在所有模型中 Recall 最低!

教学教训: Brier 单独看会误导! 必须结合 Recall 和校准曲线一起分析。
```

#### 1.3 Brier Score 的三部分分解

Brier Score 可以分解为三个组件，这正是 Murphy 分解的核心价值：

| 组件                     | 含义                         | 本实验中的表现                          |
| ---------------------- | -------------------------- | -------------------------------- |
| **鉴别力 (Refinement)**   | 模型在概率分组内的"纯度"——正类是否集中在高概率组 | 所有模型接近（0.114-0.128）              |
| **校准度 (Calibration)**  | 预测概率与实际比例之间的差距             | **差异较小**：KNN=0.0010 vs LR=0.0026 |
| **不确定性 (Uncertainty)** | 数据本身的噪声 = ȳ(1-ȳ)           | 固定值 0.2422                       |

$$
\text{Brier} = \underbrace{\text{Calibration}}_{\text{校准度}} - \underbrace{\text{Refinement}}_{\text{鉴别力}} + \underbrace{\text{Uncertainty}}\_{\text{不确定性}}
$$

校准度是我们可以 **主动改善** 的组件——这正是 **Platt Scaling、Isotonic Regression** 等方法的作用。

#### 1.4 Hosmer-Lemeshow 检验

| 模型                  | χ²              | p-value    | 校准判断 |
| ------------------- | --------------- | ---------- | ---- |
| KNN (k=15)          | 360000000083.32 | **0.0000** | ❌ 不良 |
| XGBoost             | 54.74           | **0.0000** | ❌ 不良 |
| Random Forest       | 74.64           | **0.0000** | ❌ 不良 |
| Logistic Regression | 111.46          | **0.0000** | ❌ 不良 |

**讨论：HL 检验的局限**

| 局限      | 说明                               |
| ------- | -------------------------------- |
| 对样本量敏感  | 大样本下即使微小的校准偏差也会被检测为"显著不良"        |
| 分组方式影响大 | 等频分组 vs 等距分组可能给出不同结论             |
| 只能整体检验  | 不能告诉你哪个概率区间校准差                   |
| **建议**  | 用校准曲线做**可视化诊断** + HL 检验做**形式验证** |

#### 1.5 核心讨论：高 AUC 是否一定高校准？

**答案：否。**

```
AUC 和校准度是两个独立的概念:

AUC 高 (模型排序好):                        校准良好 (概率值可信):
  正类的概率 > 负类的概率                     预测 0.8 → 80% 是正类
  (顺序正确)                                 (数值正确)

反例: 一个模型可以做到:
  - 正类概率 = [0.51, 0.52, 0.53, ...]
  - 负类概率 = [0.49, 0.48, 0.47, ...]
  → AUC = 1.00 (完美排序!)
  → 但 Brier ~ 0.25 (校准极差!)
  → 因为所有概率都集中在 0.5 附近

临床表现:
  AUC=0.90 的模型: "这个患者的风险是 0.55, 那个是 0.45"
  → 虽然排序好, 但医生无法判断"风险 0.55"是否真实

结论:
  AUC 回答"哪个患者风险更高?" 
  校准回答"风险有多高?"
  两者缺一不可。
```

### 第二部分：决策曲线分析 (DCA)

#### 2.1 DCA 的核心思想

> 传统的二分类使用 0.5 作为决策阈值——任何预测概率 > 0.5 就干预，否则不干预。但临床上，**不同疾病的决策阈值是不同的**。

```
决策阈值的临床决定因素:

阈值低 (pt=0.01):                      阈值高 (pt=0.50):
  "有点风险就干预"                        "非常确定才干预"
  ✅ 癌症筛查                            ✅ 化疗决策
  ✅ 传染病隔离                          ✅ 大型手术
  ✅ 生命危险场景                          ✅ 副作用大的治疗

  漏诊代价高 → 阈值低                     误治代价高 → 阈值高
```

DCA 通过计算**不同阈值下的净获益**，回答："这个模型在哪些阈值下比"全治 / 全不治"更有价值？"

#### 2.2 Net Benefit 公式

$$
\text{Net Benefit} = \frac{TP}{N} - \frac{FP}{N} \times \frac{p\_t}{1-p\_t}
$$

其中 $p\_t$ 是决策阈值。这个公式的含义：

- **第一项**：正确干预带来的获益
- **第二项**：错误干预带来的代价（权重随阈值增大而增大）

**当 $p\_t$ 很小时**：FP 的权重也很小 → 模型只需要避免 FN
**当 $p\_t$ 很大时**：FP 的权重也很大 → 模型必须避免 FP

#### 2.3 DCA 的三条基线

| 策略             | 含义             | 净获益曲线                |
| -------------- | -------------- | -------------------- |
| **Treat All**  | 所有患者都干预        | 高阈值时急剧下降 (因为 FP 代价大) |
| **Treat None** | 所有患者都不干预       | 始终为 0                |
| **Model**      | 仅当预测概率 > 阈值时干预 | 有一个"临床获益区间"          |

#### 2.4 实验结果

| 模型                  | 临床获益范围            | 最大净获益      | 最优阈值 | 说明            |
| ------------------- | ----------------- | ---------- | ---- | ------------- |
| Logistic Regression | \[0.01, 0.87]     | 0.4069     | 0.01 | 范围较窄          |
| Random Forest       | \[0.01, 0.97]     | 0.4072     | 0.01 | 范围宽           |
| **XGBoost**         | **\[0.01, 0.98]** | **0.4088** | 0.01 | **范围最宽、获益最大** |
| KNN (k=15)          | \[0.01, 0.94]     | 0.4073     | 0.01 | 范围宽但获益不突出     |

**核心发现**：

```
本数据集的特点:
- VIVO 占 41.15% (轻度不平衡)
- 模型对正类的预测概率分布较广
- 在 pt ≤ 0.87 时, 所有模型均能提供临床获益
- 在 pt > 0.87 后, 部分模型逐渐失去优势

临床含义:
  如果您使用的决策阈值 ≤ 0.87
  → XGBoost 是最好的选择
  → 最大净获益 0.4088 (比 Treat All 高约 0.001)

  如果您使用的决策阈值 > 0.87
  → 部分模型临床价值下降
  → 需结合具体阈值评估是否优于 "全不治"
```

#### 2.5 DCA 的传统评价"红线"

DCA 领域有一条经验规则：一个好的模型应该在**临床感兴趣的阈值范围**内净获益显著高于 Treat All 和 Treat None。通常用以下标准评价：

| 净获益提升       | 临床价值评价      |
| ----------- | ----------- |
| < 0         | 有害（模型不如不干预） |
| 0 - 0.01    | 微弱          |
| 0.01 - 0.05 | 中等          |
| > 0.05      | 显著          |

> **本实验中**，所有模型的最大净获益在 0.407 左右，属于"显著"水平。这是在轻度不平衡数据(41.15%)中的正常表现。

## 四、校准 + DCA 的完整框架

### 4.1 模型的"临床可用性"评估流程

```
第 1 步: AUC > 0.70?                                 ← 基本排除能力
  否 → 模型不可用
  是 → 继续第 2 步

第 2 步: 校准曲线接近对角线?                           ← 概率可信度
  否 → 考虑 Platt Scaling / Isotonic Regression
  是 → 继续第 3 步

第 3 步: Brier < 0.2?                                  ← 综合误差
  否 → 模型校准需要改善
  是 → 继续第 4 步

第 4 步: DCA 在临床相关阈值下 > Treat All & Treat None? ← 临床价值
  否 → 模型有统计性能但无临床价值
  是 → 模型可以用于临床决策支持

第 5 步: 确定模型的"最佳阈值范围"                        ← 临床操作化
  比如: pt ∈ [0.05, 0.15] 时有净获益
        → 预测概率 > 0.05 → 建议进一步检查
        → 预测概率 > 0.15 → 建议干预治疗
```

### 4.2 从统计到临床的"翻译"

| 统计指标                       | 对应的临床问题                                                       |
| -------------------------- | ------------------------------------------------------------- |
| AUC = 0.9168               | "这个模型有 91.68% 的概率把存活患者排在死亡患者前面"                               |
| Brier = 0.1153             | "预测概率与实际结果的平均差距约为 0.12"                                       |
| **DCA 获益范围 \[0.01, 0.98]** | **"如果你愿意接受 1%-98% 的阈值，这个模型能帮你比 '全治' 或 '全不治' 多救治约 40.9% 的患者"** |

## 五、扩展内容

### 5.1 如何改善模型的校准度？

| 方法                      | 原理                     | 优点      | 缺点            |
| ----------------------- | ---------------------- | ------- | ------------- |
| **Platt Scaling**       | 在模型输出上拟合一个 Logistic 回归 | 简单、保序   | 假设 Sigmoid 形状 |
| **Isotonic Regression** | 非参数保序回归                | 灵活、无假设  | 小样本易过拟合       |
| **Temperature Scaling** | 除以一个"温度"参数后再软最大化       | 单一参数、保序 | 最适用于神经网络      |
| **Beta Calibration**    | 专门为 \[0,1] 区间设计的校准     | 针对概率优化  | 不常见           |

```python
# Platt Scaling 示例
from sklearn.calibration import CalibratedClassifierCV

# 训练好的模型
model = LogisticRegression(class_weight='balanced')
model.fit(X_train, y_train)

# 用 Platt Scaling 校准 (在验证集上)
calibrated = CalibratedClassifierCV(model, method='sigmoid', cv=5)
calibrated.fit(X_train, y_train)
y_prob_calibrated = calibrated.predict_proba(X_test)[:, 1]
```

### 5.2 DCA 的变体

| DCA 种类       | 用法            | 临床场景                   |
| ------------ | ------------- | ---------------------- |
| **标准 DCA**   | 一个事件类型 + 一个干预 | 癌症筛查 (患癌/不患癌 → 干预/不干预) |
| **多状态 DCA**  | 多个互斥事件        | 三种治疗方案选择               |
| **生存 DCA**   | 考虑时间维度        | 5 年生存率预测               |
| **成本效益 DCA** | 用货币成本替代净获益    | 卫生经济学评价                |

## 六、总结与思考题

### 6.1 核心收获

1. **AUC 和校准度是独立的质量维度**：高 AUC 的模型可能校准差（RF: AUC=0.9162, Brier=0.1164）
2. **Brier Score 需要拆解**：Murphy 分解揭示"鉴别力+校准度+不确定性"三个组件
3. **KNN 的校准度好但 Brier 不是最低**：0.1216 的 Brier 不意味着模型最好——结合 Recall=0.8335 才能看清真相
4. **DCA 回答"模型有没有用"**：统计指标再好，如果在临床相关阈值下无净获益，模型就是无用的
5. **本数据集中所有模型在 pt ≤ 0.87 时均有临床获益**：这是由轻度不平衡的数据分布决定的
6. **DCA 已经成为医学 AI 论文的新标准**：为审稿人和临床医生提供直观的"临床价值证明"

### 6.2 思考题

1. **（基础）** 校准曲线的"完美对角线"意味着什么？如果校准线在对角线下方（例如预测 0.6 时实际只有 0.4），这说明模型是"过度自信"还是"不够自信"？医生应该相信还是调低模型的预测？
2. **（进阶）** 本教程中 KNN 有最好的校准度分量（0.0010）但 Brier 不是最低（0.1216），同时在教程 9 中 KNN 有最差的 Recall（0.8335）。解释为什么会出现这种"校准好但 Recall 差"的现象？（提示：考虑 KNN 的鉴别力分量较高，以及 Brier 的三部分分解）
3. **（进阶）** Murphy 分解中，不确定性分量是固定的（ȳ(1-ȳ)=0.2422）。如果一个数据集的不确定性更高（例如 50/50 平衡），Brier 的各部分会如何变化？校准的重要性会增加还是减少？
4. **（拓展）** DCA 的最大净获益发生在 pt=0.01（所有模型）。这意味着什么？这个阈值在临床上对应什么决策场景？为什么在 pt > 0.87 之后部分模型逐渐失去临床价值？
5. **（拓展）** 假设你开发了一个模型，AUC=0.92，Brier=0.08，HL 检验 p=0.30（良好校准）。但 DCA 显示净获益只在 \[0.03, 0.07] 范围内为正。临床科主任问你："这个模型到底能不能用？"你会如何回答？你会建议在哪些场景下使用？
6. **（实践）** 尝试在 Logistic Regression 的输出上使用 Platt Scaling（通过 `CalibratedClassifierCV(method='sigmoid')`），对比校准前后的 Brier 和校准曲线。校准改善了 Brier 吗？HL 检验的 p 值变好还是变差了？

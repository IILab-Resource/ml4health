# 案例教程 14：带特征分布的 SHAP 依赖图

## 一、前序知识要求

### 1.1 核心问题

> **SHAP 依赖图中的模式是"真实信号"还是"离群点噪声"？**

传统的 SHAP 依赖图（Dependence Plot）只展示两个维度：**特征值**（x 轴）和 **SHAP 值**（y 轴）。它没有告诉我们另一个关键信息：

- 特征值在不同区域的**样本密度**是多少？
- 如果在某个区域只有 2 个样本，但 SHAP 值剧烈变化，这种模式可信吗？

**本教程引入第三个维度——数据密度。** 通过在依赖图的底层叠加特征分布的直方图，我们可以一眼判断：哪些模式发生在数据密集区域（可信），哪些发生在数据稀疏区域（可能需要谨慎）。

### 1.2 三种图表的对比

| 图表类型 | 展示的信息 | 缺失的信息 | 本教程的改进 |
|---------|-----------|-----------|-------------|
| 标准依赖图 | 特征值 × SHAP 值 | 不知道哪些区域数据密集 | 叠加分布直方图 |
| 直方图 | 特征值分布 | 不知道 SHAP 模式 | 叠加依赖散点图 |
| **双轴依赖图** | **特征值 × SHAP 值 × 数据密度** | — | **三者合一** |

## 二、本次实践内容与目的

### 2.1 核心目的

> **为 SHAP 依赖图增加数据密度维度，评估每个特征在不同数据区域的模式可信度，使模型解释的结论更加稳健。**

### 2.2 实验设计

| 模块 | 内容 | 输出 |
|------|------|------|
| **[2]** | 双轴依赖图（2×3 网格） | Top 6 特征的整合图表 |
| **[3]** | 密度对比汇总表 | Dense Range vs Sparse Range 定量评估 |

### 2.3 风险评估规则

```
Range Ratio = SHAP Range(Sparse Region) / SHAP Range(Dense Region)

Ratio < 2:   Low Risk      → 密集区和稀疏区的 SHAP 变化接近 → 模式可信
Ratio 2-4:   Medium Risk   → 稀疏区变化略大 → 需交叉验证
Ratio > 4:   High Risk     → 稀疏区变化远大于密集区 → 极端值驱动，谨慎解读
```

## 三、模块详解

### 模块 2：双轴依赖图——如何读取

#### 2.1 图表结构

```
                  紫色半透明柱 = 特征值分布直方图

  Frequency ↑   ██
              ██  ██
              ██  ██  ██                        ● ← 散点
              ██  ██  ██  ██     ● ● ← SHAP 模式
              ██  ██  ██  ██  ██ ● ● ● ● ●
              ██  ██  ██  ██  ██ ● ● ● ● ● ●
             ──────────────────────────→ 特征值
              ● ● ● ● ● ● ● ● ●
   SHAP ↑ → ● ● ● ● ● ● ● ● ●
        ● ● ● ● ● ● ● ● ●
             ──────────────────────────→ 特征值

        密集区 (25%-75%)        稀疏区 (边缘)
```

#### 2.2 批量解读指南

> **读图顺序：密度直方图 → SHAP 散点 → 拟合曲线 → 密度标注框**

| 步骤 | 关注点 | 问题 |
|------|--------|------|
| 1 | 直方图的形状 | 数据集中在哪个区域？是否存在长尾？ |
| 2 | 密集区域的 SHAP 模式 | 直方图最高处，散点的 y 值趋势是什么？ |
| 3 | 稀疏区域的 SHAP 模式 | 边缘区域是否有剧烈变化？ |
| 4 | 拟合曲线的弯曲位置 | 弯曲发生在密集区还是稀疏区？ |
| 5 | 密度标注框的 Ratio | 量化评估——稀疏/密集的比例是否合理？ |

#### 2.3 本数据集的 6 个特征评估

| 特征 | Importance | R²_quad | Dense Range | Sparse Range | **Ratio** | Risk | 解读 |
|------|-----------|---------|-------------|--------------|-----------|------|------|
| **Diagnostic.means** | 0.2421 | 0.9518 | 0.7619 | 0.0979 | **0.13** | **Low** | ✅ 密集区 SHAP 变化远大于稀疏区，模式可信 |
| **year** | 0.1274 | 0.7618 | 0.4222 | 0.4616 | **1.09** | **Low** | ✅ 跨越全数据范围一致可信 |
| Raca.Color | 0.0421 | 0.8042 | 0.2421 | 0.1673 | 0.69 | Low | 密集区模式为主 |
| **Extension** | 0.0330 | 0.8298 | 0.0477 | 0.5252 | **11.02** | **⚠️ High** | 密集区几乎无变化，稀疏区剧烈变化 |
| Code.Profession | 0.0287 | 0.8028 | 0.1851 | 0.1782 | 0.96 | Low | 全范围一致可信 |
| Age | 0.0280 | 0.7871 | 0.1163 | 0.2462 | 2.12 | Medium | 边缘区波动略大 |

#### 2.4 教学陷阱 1：Extension 的 Ratio=11.02

```
Ratio = 11.02 意味着什么？

Dense Range (0.048): 在数据密集的中间 50% 区域，SHAP 值几乎水平
  → 在这个区域，Extension"不重要"
  → SHAP 值在 -0.02 到 +0.02 之间波动

Sparse Range (0.525): 在数据稀疏的边缘区域，SHAP 值剧烈变化
  → 少数几个极端值的 Extension 样本贡献巨大
  → SHAP 值从 -0.25 到 +0.28

教学结论:
  1. Extension 在常规范围内几乎不影响预测
  2. 但在某些特殊值上（可能对应极少数扩展类型），影响非常大
  3. 这种模式可能是真实的（某些罕见扩展方式预后极差/极好）
  4. 但也可能受到少数离群点的影响——需要验证
  5. 验证方法: 去掉极端值后重新计算 AUC
```

#### 2.5 教学陷阱 2：高 R² ≠ 可信赖

```
对比 Extension 和 Diagnostic.means:

Extension:
  R²_quad = 0.8298 → 高
  Ratio = 11.02    → 高风险

Diagnostic.means:
  R²_quad = 0.9518 → 更高
  Ratio = 0.13     → 低风险

关键认知:
  R² 高只说明"二次曲线拟合得好"
  但"拟合得好"可能是因为少数极端值的"杠杆效应"
  极端值拉动了拟合曲线，使 R² 出现"虚假高"

所以:
  高 R² + 低 Ratio = 可信的非线性模式
  高 R² + 高 Ratio = 被极端值驱动的模式
```

## 四、对模型解释实践的影响

### 4.1 临床解读中的"密度提醒"

```
"医生, 我们的模型认为 year 是重要预测因素,
 而且这个模式在全数据范围内是一致的。"

比下面这个说法更可信：

"医生, 我们的模型认为 Diagnostic.means 很重要,
 但这个结论主要基于少数罕见诊断方式,
 在常规诊断范围内的影响其实很小。"
```

### 4.2 论文中对密度风险的讨论模板

```
在论文的"局限性"部分：

"虽然 SHAP 分析揭示了个别特征存在非线性交互模式
（如 Diagnostic.means, Ratio=17.17），
但这些模式主要集中在数据稀疏区域，
可能受到离群值的较大影响。
因此在解读这些特征时，我们聚焦于
数据密集区（25%-75% 分位数）的 SHAP 变化趋势。
建议在更大样本量的独立数据集中验证这些边缘模式。"
```

### 4.3 特征工程的优先级调整

```
基于 Ratio 的特征工程建议:

Low Ratio (year):
  无需特殊处理 → 自然可信

Medium Ratio (Code.Profession, Age, Raca.Color):
  可以做 Winsorization (截尾处理)
  限制极端值的影响而不损失主流信息

High Ratio (Diagnostic.means, Extension):
  建议:
  1. 检查极端值是否真的是"合法"的医学编码
  2. 如果合法 → 保持, 但解释时注明"边缘模式"
  3. 如果错误 → 修正或删除
  4. 考虑分箱处理 → 将极端值合并到"其他"类别
```

### 4.4 离群值检测的协同效应

结合教程 1（EDA 离群值分析）和本教程：

```
教程 1 发现:
  IQR 和 Z-score 检测到 Age 有 1% 左右的离群值

本教程发现:
  Age 的 Ratio=2.12 → 中等风险

协同结论:
  虽然在 Age 中检测到离群值，
  但这些离群值对 SHAP 模式的"扭曲"是有限的 (Ratio=2.12)。
  可接受。

但如果某个特征同时满足:
  - IQR 检测到大量离群值 (>5%)
  - Ratio > 4
  - 并且该特征被 Boruta 确认为 Confirmed

  则: 该特征的模型重要性可能被离群值"人为抬高"。
  建议: 截尾后重新训练模型 → 比较 AUC 变化。
```

## 五、扩展内容

### 5.1 Ratio 的统计检验

Ratio = Sparse Range / Dense Range 本身统计意义有限。更严格的检验是：

```python
from scipy.stats import mannwhitneyu

# 密集区 vs 稀疏区的 SHAP 绝对值是否显著不同?
stat, p = mannwhitneyu(
    np.abs(shap_vals[dense_mask]),
    np.abs(shap_vals[sparse_mask])
)
# p < 0.05 → 两个区域的 SHAP 绝对值的分布显著不同
# 这定量证明了"稀疏区模式确实不一样"
```

### 5.2 不同分位数定义的敏感性

本教程使用 25%-75% 定义"密集区"。可以使用不同的分位数边界：

| 定义 | 密集区宽度 | 敏感性 | 建议 |
|------|-----------|--------|------|
| 20%-80% | 60% 样本 | 稳健但保守 | 大样本 (n > 1000) |
| **25%-75%** | **50% 样本** | **标准** | **默认推荐** |
| 30%-70% | 40% 样本 | 严格 | 小样本 (n < 500) |

### 5.3 本方法的局限性

| 局限性 | 原因 | 缓解 |
|--------|------|------|
| 分位数边界主观 | 25%-75% 是人为定义 | 使用多个分位数定义做敏感性分析 |
| 稀疏区的"信号"可能是真的 | 某些罕见但重要的医学模式 | 与临床专家确认极端值的医学合理性 |
| Ratio 忽略方向性 | 只比较范围，不比较趋势方向 | 结合依赖图目视检查 |
| 小样本依赖 | 500 个样本的密集/稀疏划分可能不稳健 | 增加 SHAP 样本数或引导 |

### 5.4 从 Ratio 到"特征可信度"的映射

```
            Ratio < 2       2 ≤ Ratio ≤ 4       Ratio > 4
           ┌─────────────────────────────────────────
重要性高   │  黄金特征        慎重解读             需验证
重要性中   │  可靠特征        注意边缘             边缘主导
重要性低   │ 不重要特征       不重要特征           不重要特征
```

**教学要点**：如果一个特征重要性低（如 Extension，0.033）且 Ratio 高（11.02），它的重要性可能被少数极端值"虚假抬升"了。

## 六、总结与思考题

### 6.1 核心收获

1. **Diagnostic.means 是"全范围可信"的特征**（Ratio=0.13）——它在数据密集区的 SHAP 变化远大于稀疏区，模式由密集数据驱动
2. **Extension 的 Ratio=11.02**——这是一个极端的教学示例，展示了"密集区几乎平坦，稀疏区剧烈变化"的模式
3. **高 R² 不一定代表可信赖**——如果 R² 来自稀疏区极端值的杠杆效应，R² 是"虚假高"（Extension 的 R²=0.8298 但 Ratio=11.02）
4. **数据密度提供了模型解释的"置信度"维度**——每次看到依赖图都要问："这个模式发生在数据集中还是边缘？"
5. **Ratio > 4 的特征需要谨慎**——这些特征的边缘模式可能不合理地影响了整体的解释
6. **Ratio < 1 也是好信号**——当 Dense Range > Sparse Range 时（如 Diagnostic.means 的 Ratio=0.13），说明 SHAP 模式集中在数据密集区，这正是我们期望的

### 6.2 思考题

1. **（基础）** 解释为什么 Extension 的 R²_quad=0.8298 但 Ratio=11.02。高 R² 和低 Ratio 如何共存？

2. **（进阶）** 如果将一个特征的极端值全部替换为中位数，然后重新运行 SHAP，你认为该特征的 Ratio 会升高还是降低？为什么？

3. **（进阶）** 本教程用 25%-75% 定义"密集区"。如果改用 10%-90% 定义，"密集区"范围变大但"密集"程度降低。你认为这会如何影响 Ratio？试设计实验验证。

4. **（拓展）** Ratio 的另一个替代度量是"稀疏区的样本占 SHAP 总方差的百分比"。计算每个特征的这个比例，并与 Ratio 比较。哪个更能反映"极端值驱动"的问题？

5. **（拓展）** 本教程发现 Extension 的 Ratio=11.02 但重要性只有 0.033。解释为什么一个重要性很低的特征会具有高 Ratio 值？这说明了什么关于 SHAP 依赖图的解读？

6. **（实践）** 对 Extension 做截尾处理（Winsorization，将上下 5% 的极端值替换为边界值）。比较截尾前后的 SHAP 模式（依赖图 + Ratio）：Ratio 变化了多少？模型的 AUC 变化了多少？

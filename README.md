# ml4health
面向南京医科大学卓越工程师初阶学习的医学机器学习教程项目，覆盖从探索性数据分析、统计检验到高级模型解释和因果推断的完整 ML 流程，配有详细的教学文档和可复现的 Python 代码。

## 教程模块

| 模块  | 主题                      | 能做什么（成果导向）                                 | 一句话能力标签         |
| --- | ----------------------- | ------------------------------------------ | --------------- |
| 01  | 探索性数据分析 EDA             | 像医生做体检一样快速了解数据健康状况，发现异常、规律和潜在问题，让数据“开口说话”。 | 看懂数据            |
| 02  | 统计分析                    | 判断两个组到底有没有真正差异，而不是“看起来好像不同”，让结论有统计学依据。     | 找到真正有意义的差异      |
| 03  | 数据预处理与缺失值插补             | 把杂乱、不完整的数据整理成模型喜欢的样子，大幅提升分析和建模质量。          | 把杂乱数据变成可分析数据    |
| 04  | 特征工程                    | 把普通变量加工成更有价值的信息，让模型发现隐藏规律、获得更好的预测能力。       | 提升模型效果          |
| 05  | 特征选择                    | 从海量变量中找到真正重要的因素，构建更简洁、更稳定、更容易发表的模型。        | 找到关键影响因素        |
| 06  | 降维与聚类                   | 发现数据中的天然人群分层和隐藏模式，找到“哪些患者其实属于同一种类型”。       | 发现隐藏人群          |
| 07  | 数据泄漏分析                  | 避免模型“作弊拿高分”，保证结果真实可靠，避免论文返修和项目翻车。          | 避免模型踩坑          |
| 08  | 交叉验证                    | 学会科学评估模型性能，不再因为一次随机划分而得到偶然结果。              | 让结果更加可靠         |
| 09  | 建模对比                    | 能够独立完成从传统统计模型到热门机器学习模型的构建与比较。              | 建立自己的预测模型       |
| 10  | 类别不平衡处理                 | 即使阳性病例很少，也能训练出真正有识别能力的模型。                  | 解决少数病例难题        |
| 11  | 校准分析与决策曲线 DCA           | 不仅知道模型准不准，还知道模型是否真的值得用于临床决策。               | 判断模型是否值得应用      |
| 12  | 模型解释（SHAP + LIME）       | 打开机器学习“黑箱”，清楚解释模型为什么做出这个预测。                | 解释模型为什么这样判断     |
| 12b | 高级 SHAP 可视化             | 制作高级解释图和高质量论文图表，让结果既专业又直观。                 | 制作高水平论文图表       |
| 13  | SHAP 交互效应分析             | 发现变量之间如何相互影响，而不仅仅是谁更重要。                    | 发现变量之间的联合作用     |
| 14  | SHAP 依赖图与数据密度           | 判断模型结论在哪些数据区域更可信，提升研究结果的说服力。               | 识别哪些结论最可信       |
| 15  | SHAP 聚类分析               | 发现不同风险模式的人群，为精准医疗和分层管理提供依据。                | 找到不同类型患者        |
| 16  | SHAP 决策路径分析             | 像阅读病例一样，逐步追踪模型如何得出某位患者的预测结果。               | 追踪单个患者的预测过程     |
| 17  | SHAP 稳定性 Bootstrap 分析   | 验证模型解释是否稳定可靠，而不是一次训练得到的偶然发现。               | 验证结果是否稳定        |
| 18  | 异质性处理效应 HTE（双重机器学习 DML） | 回答“什么人从治疗中获益更多”这样的高级临床问题，迈向因果推断与精准医疗。      | 回答“谁更受益”这样的临床问题 |

### 分级学习

本教程适合不同基础的学生。每个案例内部按知识点难度分为三个等级：

- **L1 核心概念与实践**：零基础入门，掌握核心概念，逐行理解代码
- **L2 方法深入**：理解方法原理，参数调优，不同方法对比选择
- **L3 理论扩展**：数学理论、扩展内容、论文级分析方法

详细分级说明、学习节奏、技能树 → [LEARNING_PATH.md](LEARNING_PATH.md)

## 学习路径概览

整个教程分为 4 个阶段，每个阶段完成后都有对应的综合实践任务（详见 [TASKS.md](TASKS.md)）：

| 阶段 | 模块 | 核心内容 | 综合节点 |
|------|------|---------|---------|
| **基础入门** | 01-02 | EDA + 统计分析 | Node 1：完整 EDA 报告 |
| **特征工程** | 03-05 | 预处理 → 特征构造 → 特征选择 | Node 2：完整流水线 |
| **建模评估** | 06-11 | 降维 → 防泄漏 → CV → 建模 → 不平衡 → DCA | Node 3-4：审计 + 端到端 |
| **可解释性 & 因果** | 12-18 | SHAP 全系列 + 双重机器学习 DML | Node 5-7：SHAP 报告 + 可视化系统 |

# 案例教程 18：异质性处理效应 (HTE) — 双重机器学习 (DML) 方法

## 一、前序知识要求

### 1.1 核心问题

> **癌症患者的生存率不仅取决于"是否转移"，更取决于"什么样的人发生了转移"。不同年龄段、不同性别、不同种族的患者，转移对生存的影响是一样的吗？**

本教程探讨的核心因果问题：

```
因果问题: 癌症转移 (T) → 对生存 (Y) 的因果效应
  T=1: 已发生转移 (METÁSTASE)
  T=0: 局限期 (LOCALIZADO)
  Y:   存活状态 (1=存活, 0=死亡)

目标: 这个效应在不同人群之间是否不同？
      → 这就是"异质性处理效应" (HTE)
      → 也叫"条件平均处理效应" (CATE)
```

### 1.2 为什么需要双重机器学习 (DML)？

在医学观察性研究中，处理分配（是否发生转移）通常不是随机的：

```
混淆问题:

  年龄较大 → 更容易转移    ← 年龄同时影响转移和生存
  年龄较大 → 更可能死亡    

  如果直接用"转移"和"生存"的相关性:
    相关 ≠ 因果
    因为年龄同时影响了两者

  标准方法 (如倾向得分匹配) 的局限:
    当变量很多时 (高维混淆)，匹配困难
    使用复杂 ML 模型时，容易过拟合
```

**双重机器学习 (DML) 解决的正是这个问题：**

```
DML 的核心创新 — 交叉拟合 (Cross-Fitting):

  第 1 步: 将数据分成 K 折 (例如 5 折)

  第 2 步: 对每一折 i:
    - 用其他 K-1 折数据训练:
      结果模型:  E[Y|X, W]   ← 预测存活
      倾向模型:  P(T=1|X, W) ← 预测转移
    - 在第 i 折上计算"残差":
      Y_res = Y - E[Y|X, W]    ← 实际存活 - 预测存活
      T_res = T - P(T=1|X, W)  ← 实际转移 - 预测转移

  第 3 步: 对残差做回归:
    Y_res ~ T_res  ← 这个斜率就是 CATE

  为什么有效?
    - 残差中去除了混淆因素的影响
    - 交叉拟合避免了过拟合偏差
    - 具有半参数有效性 (理论上的最优效率)
```

### 1.3 DML 与非 DML 方法的对比

| 方法               | 对高维混淆的鲁棒性 | 需指定函数形式 | 理论性质      |
| ---------------- | --------- | ------- | --------- |
| 线性回归             | 弱         | 是       | 需线性假设     |
| 倾向得分匹配           | 中等        | 否       | 匹配偏差      |
| 工具变量 (IV)        | 强         | 是       | 需有效工具     |
| **DML**          | **强**     | **否**   | **半参数有效** |
| 元学习器 (T-Learner) | 中等        | 否       | 需额外校正     |

### 1.4 前序知识

- **因果推断基础**：处理、结果、混淆变量的概念
- **随机森林**：集成学习的基本原理
- **Lasso 回归**：L1 正则化与特征选择
- **交叉验证**：训练/测试集划分的基本概念
- **K-Means 聚类**：无监督学习的基本方法

## 二、本次实践内容与目的

### 2.1 核心目的

> **使用 DROrthoForest 和 CausalForestDML 两种先进的 DML 模型，估计癌症转移对生存的异质性处理效应 (CATE)，并通过聚类分析发现具有不同效应模式的子群体。**

### 2.2 使用的模型

#### 模型 1：DROrthoForest (Double Robust Orthogonal Random Forest)

```
核心思想:
  DROrthoForest = 正交随机森林 + 双重稳健性

  正交化 (Orthogonalization):
    将"效应估计"与"讨厌参数估计"分离
    → 效应估计对模型误差不敏感

  双重稳健性 (Double Robustness):
    只要"结果模型"或"倾向得分模型"中至少一个正确
    → 效应估计就是一致的 (渐近无偏)
    → 这在实践中非常重要，因为我们很少能确保完全正确

  实现方式:
    每棵树在 Bootstrap 样本上训练
    叶子节点内进行局部 DML 估计
    结果聚合所有树的输出
```

#### 模型 2：CausalForestDML (Causal Forest DML)

```
核心思想:
  CausalForestDML = 因果森林 + 双重机器学习

  不同于普通随机森林:
    分裂准则: 最大化处理效应的异质性 (而非 MSE)
    叶子节点: 估计局部平均处理效应 (LATE)
    聚合方式: 加权平均 (而非简单平均)

  与 DROrthoForest 的区别:
    更轻量, 训练更快
    内置交叉拟合
    支持更多的推断功能 (置信区间、p 值)
```

### 2.3 分析流程

```
数据准备:
  Y = 存活状态 (target)
  T = 是否转移 (来自 Extension: LOCALIZADO=0, METÁSTASE=1)
  X = 异质性特征 (Age)      ← 假设效应随年龄变化
  W = 控制变量 (Gender, Raca.Color, year, Laterality, Diagnostic.means)

阶段 1 — DROrthoForest:
  训练 → 估计 CATE(x) → 计算 95% 置信区间

阶段 2 — CausalForestDML:
  训练 → 估计 CATE(x) → 计算 95% 置信区间

阶段 3 — 模型比较:
  并排可视化两个模型的 CATE 曲线和置信区间

阶段 4 — 高级推断:
  恒定边际效应: 假设效应不随 X 变化
  处理效应推断: 估计 CATE(x) 并提供 p 值和置信区间

阶段 5 — 聚类分析:
  基于 CATE 估计值 + 特征 → K-Means 聚类
  发现不同处理效应模式的子群体
```

## 三、结果解读指南

### 3.1 如何阅读模型比较图

```
左图: DROrthoForest   右图: CausalForestDML

X 轴: Age (标准化后)
Y 轴: 处理效应 (CATE)

橙色/绿色曲线: 估计的 CATE(x)
半透明区域: 95% 置信区间

解读要点:
  1. 曲线趋势:
     - 如果曲线向下倾斜: 年龄越大, 转移的负效应越强
     - 如果曲线向上倾斜: 年龄越大, 转移的负效应越弱
     - 如果曲线平坦: 处理效应不存在年龄异质性
  
  2. 置信区间宽度:
     - 区间越窄: 该年龄段的效应估计越精确
     - 区间越宽: 不确定性越大 (可能是样本量少或噪声大)

  3. 模型一致性:
     - 两种模型的曲线趋势一致吗?
     - 如果趋势矛盾, 可能说明估计不可靠
```

### 3.2 如何理解 ATE 与 CATE 的区别

```
ATE (平均处理效应):
  "总体而言, 转移使存活概率降低约 42-53%"
  → 一个简单的数字, 但掩盖了人群差异

CATE (条件平均处理效应):
  "对于年轻患者, 转移的效应约为 -X%"
  "对于老年患者, 转移的效应约为 -Y%"
  → 揭示了差异, 对个性化决策更有价值

为什么 CATE 比 ATE 更重要?
  如果中老年患者的风险远高于青年
  → 筛查和干预策略应针对中老年人群
  → 这就是"精准医学"的核心思想
```

### 3.3 恒定边际效应 vs 异质性效应

```
恒定边际效应:
  假设: 处理效应对所有患者都一样
  结果: 一条水平线 (例如 -0.58)
  优点: 简单、易解释
  缺点: 可能过度简化

异质性效应 (CATE):
  假设: 处理效应随特征变化
  结果: 一条曲线
  优点: 更丰富的信息
  缺点: 更复杂, 需要谨慎解读不稳定性

课堂讨论:
  什么时候应该用恒定效应? (政策制定)
  什么时候应该用异质性效应? (个性化决策)
```

### 3.4 聚类分析的医学意义

```
基于 CATE 的聚类结果:
  将患者分为 K 组, 每组在年龄和转移效应上有独特模式

例子 (实际 K=6):
  集群 A: 高龄 + 强负效应 → 最需要早期筛查
  集群 B: 中龄 + 中等负效应 → 常规管理
  集群 C: 低龄 + 弱负效应 → 预后相对较好
  集群 D-F: 其他模式 → 需要进一步分析

对临床决策的帮助:
  识别"高风险"人群 (转移效应最强)
  为不同人群制定差异化的随访和干预方案
  发现可能需要额外研究的新型亚群
```


### 4.4 工具变量 (IV) 与 DML 的结合

如果存在可信的工具变量 Z（影响处理但不直接影响结果），DML 也能整合：

```
DML + IV 的优势:
  进一步减少未观测混淆的偏倚
  在观察性研究中提供更稳健的因果估计
  适用场景: 存在自然实验、政策变化等
```

### 4.5 敏感性分析

对于 DML 估计的 CATE，可以进行敏感性分析：

```
1. 改变 lambda_reg
   正则化强度影响模型复杂度 → 观察 CATE 是否稳定

2. 改变 subsample_ratio
   每棵树的样本比例 → 影响方差-偏差权衡

3. 改变 n_estimators
   树的数量 → 评估稳定性

4. 去除部分 W 变量
   观察 CATE 是否对控制变量的选择敏感
```

### 4.6 反事实预测

DML 训练之后，可以进一步做反事实预测：

```python
# 假设所有患者都接受处理 → 预期结局
Y_cf_1 = model.predict(X, W, T=1)

# 假设所有患者都不接受处理 → 预期结局
Y_cf_0 = model.predict(X, W, T=0)

# 个体处理效应 (ITE)
ITE = Y_cf_1 - Y_cf_0
```

### 4.7 政策学习 (Policy Learning)

基于估计的 CATE，可以设计最优干预策略：

```
最优政策分配:
  如果 CATE(x) > 阈值 → 推荐干预
  如果 CATE(x) < 阈值 → 不干预

目标: 在有限资源下最大化整体收益
```

## 五、常见问题

### 5.1 DML 的结果如何与常规 Logistic Regression 比较？

```
常规 Logistic Regression:
  logit(P(Y=1)) = β₀ + β₁T + β₂X + β₃W
  → β₁ 是"调整后的 OR"
  → 假设效应是常数, 线性, 无交互

DML:
  CATE(x) = E[Y(1)-Y(0) | X=x]
  → 效应可以随 X 变化
  → 不假设线性
  → 对模型设定错误更鲁棒

如果两个结果矛盾:
  通常是 Logistic Regression 的假设被违反
  DML 通常更可靠
```

### 5.2 样本量需要多大？

```
经验法则:
  DROrthoForest: 至少 1000+ 样本
  CausalForestDML: 至少 500+ 样本
  更多的 W 变量 → 需要更多的样本

本教程使用 3000 样本, 5 个 W 变量 → 足够的统计效力
```

### 5.3 ATE 为负但 CI 包含 0 怎么办？

```
说明: 
  平均效应可能是 0, 但不排除存在异质性
  此时 CATE 分析更有价值:
    某些子群体可能有显著效应
    另一些可能确实没有效应

应对策略:
  1. 检查是否遗漏了重要 W 变量
  2. 增加样本量提高统计效力
  3. 使用更灵活的模型 (更多树, 更小叶子)
```

### 5.4 CATE 置信区间很宽怎么办？

```
可能原因:
  1. 样本量不足
  2. 该子群体中样本太少
  3. 噪声太大 (Y 的方差大)
  4. 处理分配不均衡

解决方法:
  1. 增加样本量
  2. 增大 min_leaf_size (减少过拟合)
  3. 增加 n_estimators (降低方差)
  4. 尝试用 BLB 自助法 (宽 CI 的"诚实"反映)
```

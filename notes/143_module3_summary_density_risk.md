## 一、三个特征类别的教学分组

根据 Ratio 值，把 6 个特征分成三个类别：

### 1.1 类别 1：全范围一致型（Ratio < 2）

```
类别 1: 全范围一致型 (Ratio < 2)
  year (Ratio=1.71)
  → 可信度最高
  → SHAP 模式在数据密集区和稀疏区一致
  → 对 year 的解读可以直接用于临床讨论
```

**year** 是唯一属于这一类别的特征：

- Importance = 0.2690（最高）
- R² = 0.8595（高）
- Dense Range = 0.5087，Sparse Range = 0.8676
- Ratio = 1.71（< 2）

**解读**：year 的 SHAP 模式在全数据范围一致可信。无论患者是哪一年诊断的，year 对预测的影响模式都成立。这是"黄金特征"——重要性高、非线性明显、模式在全数据范围一致。

### 1.2 类别 2：边缘微调型（Ratio 2-4）

```
类别 2: 边缘微调型 (Ratio 2-4)
  Code.Profession (3.51), Age (3.03), Raca.Color (3.52)
  → 主流模式可信，边缘调整需确认
  → 解释时聚焦在密集区域的模式
```

这一类别有 3 个特征：

- **Code.Profession**：Ratio = 3.51
- **Age**：Ratio = 3.03
- **Raca.Color**：Ratio = 3.52

**解读**：这些特征的主流模式（密集区）可信，但边缘区域（稀疏区）的 SHAP 变化是密集区的 3 倍左右。解释时应该聚焦在密集区域的模式，对边缘区域的解读需要谨慎。

### 1.3 类别 3：边缘主导型（Ratio > 4）

```
类别 3: 边缘主导型 (Ratio > 4)
  Diagnostic.means (17.17), Extension (5.16)
  → 稀疏区的极端值可能驱动了整体模式
  → 必须分层验证
  → 论文中需要讨论"数据稀疏性"限制
```

这一类别有 2 个特征：

- **Diagnostic.means**：Ratio = 17.17（极端）
- **Extension**：Ratio = 5.16

**解读**：这些特征的 SHAP 模式主要由稀疏区的极端值驱动。在密集区，SHAP 值几乎不变；在稀疏区，SHAP 值剧烈变化。这种模式可能是真实的（某些罕见诊断方式预后极差/极好），但也可能受到少数离群点的影响。必须分层验证，论文中需要讨论"数据稀疏性"限制。

## 二、三维评估框架

将本教程的 Ratio 与教程 12b 的 R²、教程 13 的交互强度组合，形成完整的特征评估框架：

| 特征               | 重要性 | 非线性 (ΔR²) | 交互强度 | **密度风险 (Ratio)** | 综合诊断       |
| ---------------- | --- | --------- | ---- | ---------------- | ---------- |
| year             | 极强  | 强         | 高    | **低**            | ✅ 第一可信特征   |
| Diagnostic.means | 中   | 极强        | 高    | **极高**           | ⚠️ 模式可能不可靠 |
| Code.Profession  | 中   | 中         | 中    | **中**            | 需交叉验证      |
| Age              | 中   | 中         | 中    | **中**            | 主流模式可信     |
| Extension        | 低   | 弱         | 中    | **高**            | 注意边缘影响     |
| Raca.Color       | 低   | 忽略        | 中    | **中**            | 基本可信       |

### 2.1 框架的四个维度

1. **重要性（mean |SHAP|）**：特征对预测的平均贡献大小。
2. **非线性（ΔR²）**：来自教程 12b，二次 R² - 线性 R²，衡量非线性程度。
3. **交互强度**：来自教程 13，特征与其他特征的 SHAP 值相关系数。
4. **密度风险（Ratio）**：本教程的核心指标，衡量 SHAP 模式在密集区与稀疏区的一致性。

### 2.2 综合诊断

- **year**：四维都优秀（重要性极强、非线性强、交互高、密度风险低）→ 第一可信特征。
- **Diagnostic.means**：重要性中、非线性极强、交互高，但密度风险极高 → 模式可能不可靠，需要分层验证。
- **Code.Profession**：四维都中等 → 需交叉验证。
- **Age**：四维都中等 → 主流模式可信。
- **Extension**：重要性低、非线性弱、交互中、密度风险高 → 注意边缘影响。
- **Raca.Color**：重要性低、非线性忽略、交互中、密度风险中 → 基本可信。

> 💡 **三维评估框架的价值**
>
> 单看任何一个维度都不够：
>
> - 只看重要性 → 不知道模式是否可信。
> - 只看 R² → 不知道 R² 是否来自极端值。
> - 只看交互强度 → 不知道交互是否在密集区。
> - 只看 Ratio → 不知道特征是否重要。
>
> 四维组合才能给出完整的特征诊断。

## 三、对模型解释实践的影响

### 3.1 临床解读中的"密度提醒"

```
"医生, 我们的模型认为 year 是重要预测因素,
 而且这个模式在全数据范围内是一致的。"

比下面这个说法更可信：

"医生, 我们的模型认为 Diagnostic.means 很重要,
 但这个结论主要基于少数罕见诊断方式,
 在常规诊断范围内的影响其实很小。"
```

**教学要点**：在临床解读中，应该明确区分"全范围可信"和"边缘主导"的特征。前者可以直接用于临床讨论，后者需要注明"边缘模式"。

### 3.2 论文中对密度风险的讨论模板

```
在论文的"局限性"部分：

"虽然 SHAP 分析揭示了个别特征存在非线性交互模式
（如 Diagnostic.means, Ratio=17.17），
但这些模式主要集中在数据稀疏区域，
可能受到离群值的较大影响。
因此在解读这些特征时，我们聚焦于
数据密集区（25%-75% 分位数）的 SHAP 变化趋势。
建议在更大样本量的独立数据集中验证这些边缘模式。"
```

**教学要点**：审稿人关心"模式可信度"问题。在论文中主动讨论数据稀疏性，能体现研究的严谨性。

### 3.3 特征工程的优先级调整

```
基于 Ratio 的特征工程建议:

Low Ratio (year):
  无需特殊处理 → 自然可信

Medium Ratio (Code.Profession, Age, Raca.Color):
  可以做 Winsorization (截尾处理)
  限制极端值的影响而不损失主流信息

High Ratio (Diagnostic.means, Extension):
  建议:
  1. 检查极端值是否真的是"合法"的医学编码
  2. 如果合法 → 保持, 但解释时注明"边缘模式"
  3. 如果错误 → 修正或删除
  4. 考虑分箱处理 → 将极端值合并到"其他"类别
```

### 3.4 离群值检测的协同效应

结合教程 1（EDA 离群值分析）和本教程：

```
教程 1 发现:
  IQR 和 Z-score 检测到 Age 有 1% 左右的离群值

本教程发现:
  Age 的 Ratio=3.03 → 中等风险

协同结论:
  虽然在 Age 中检测到离群值，
  但这些离群值对 SHAP 模式的"扭曲"是有限的 (Ratio=3.03)。
  可接受。

但如果某个特征同时满足:
  - IQR 检测到大量离群值 (>5%)
  - Ratio > 4
  - 并且该特征被 Boruta 确认为 Confirmed

  则: 该特征的模型重要性可能被离群值"人为抬高"。
  建议: 截尾后重新训练模型 → 比较 AUC 变化。
```

## 四、关键认知：高 R² ≠ 可信赖

### 4.1 对比 Diagnostic.means 和 year

```
对比 Diagnostic.means 和 year:

Diagnostic.means:
  R²_quad = 0.7638 → 高
  Ratio = 17.17    → 高风险

year:
  R²_quad = 0.8595 → 更高
  Ratio = 1.71     → 低风险

关键认知:
  R² 高只说明"二次曲线拟合得好"
  但"拟合得好"可能是因为少数极端值的"杠杆效应"
  极端值拉动了拟合曲线，使 R² 出现"虚假高"

所以:
  高 R² + 低 Ratio = 可信的非线性模式
  高 R² + 高 Ratio = 被极端值驱动的模式
```

### 4.2 杠杆效应的直觉理解

想象一根杠杆：

- 支点在中间（密集区）。
- 一端放一个重物（极端值）。
- 重物会拉动杠杆，使支点（密集区）的拟合曲线被"撬动"。

在统计中，极端值（高杠杆点）对拟合曲线的影响远大于密集区的点。即使密集区的 SHAP 值几乎不变，少数极端值也能让 R² 看起来很高。

### 4.3 如何识别"虚假高 R²"

| 指标          | 可信的高 R²           | 虚假的高 R²          |
| ----------- | ----------------- | ---------------- |
| R²          | 高                 | 高                |
| Ratio       | 低（< 2）            | 高（> 4）           |
| Dense Range | 与 Sparse Range 接近 | 远小于 Sparse Range |
| 拟合曲线弯曲位置    | 在密集区              | 在稀疏区             |
| 极端值数量       | 少                 | 多                |

## 五、从 Ratio 到"特征可信度"的映射

```
            Ratio < 2       2 ≤ Ratio ≤ 4       Ratio > 4
           ┌─────────────────────────────────────────
重要性高   │  黄金特征        慎重解读             需验证
重要性中   │  可靠特征        注意边缘             边缘主导
重要性低   │ 不重要特征       不重要特征           不重要特征
```

**教学要点**：如果一个特征重要性低（如 Extension，0.011）且 Ratio 高（5.16），它的重要性可能被少数极端值"虚假抬升"了。

### 5.1 四种典型组合

| 组合             | 例子               | 解读                    |
| -------------- | ---------------- | --------------------- |
| 高重要性 + 低 Ratio | year             | **黄金特征**——重要且可信       |
| 高重要性 + 高 Ratio | Diagnostic.means | **需验证**——重要但模式可能不可靠   |
| 低重要性 + 低 Ratio | （本数据集无）          | **不重要特征**——不重要但模式可信   |
| 低重要性 + 高 Ratio | Extension        | **虚假抬升**——重要性可能被极端值抬高 |

## 六、本方法的局限性

| 局限性           | 原因                   | 缓解               |
| ------------- | -------------------- | ---------------- |
| 分位数边界主观       | 25%-75% 是人为定义        | 使用多个分位数定义做敏感性分析  |
| 稀疏区的"信号"可能是真的 | 某些罕见但重要的医学模式         | 与临床专家确认极端值的医学合理性 |
| Ratio 忽略方向性   | 只比较范围，不比较趋势方向        | 结合依赖图目视检查        |
| 小样本依赖         | 500 个样本的密集/稀疏划分可能不稳健 | 增加 SHAP 样本数或引导   |

### 6.1 不同分位数定义的敏感性分析

本教程使用 25%-75% 定义"密集区"。可以使用不同的分位数边界：

| 定义          | 密集区宽度      | 敏感性    | 建议             |
| ----------- | ---------- | ------ | -------------- |
| 20%-80%     | 60% 样本     | 稳健但保守  | 大样本 (n > 1000) |
| **25%-75%** | **50% 样本** | **标准** | **默认推荐**       |
| 30%-70%     | 40% 样本     | 严格     | 小样本 (n < 500)  |

### 6.2 Ratio 的统计检验扩展

Ratio = Sparse Range / Dense Range 本身统计意义有限。更严格的检验是：

```python
from scipy.stats import mannwhitneyu

# 密集区 vs 稀疏区的 SHAP 绝对值是否显著不同?
stat, p = mannwhitneyu(
    np.abs(shap_vals[dense_mask]),
    np.abs(shap_vals[sparse_mask])
)
# p < 0.05 → 两个区域的 SHAP 绝对值的分布显著不同
# 这定量证明了"稀疏区模式确实不一样"
```

## 案例教程 14 总结

整个案例教程 14「带特征分布的 SHAP 依赖图」到此结束。回顾四个模块：

**核心收获**：

1. **第三个维度"数据密度"**：本教程的核心创新是在 SHAP 依赖图上叠加特征分布直方图，让读者一眼看出"SHAP 模式发生在数据密集区还是稀疏区"。
2. **Ratio 指标**：`Ratio = Sparse Range / Dense Range`，量化"稀疏区 SHAP 变化是密集区的几倍"。
   - Ratio < 2：模式可信。
   - Ratio 2-4：需交叉验证。
   - Ratio > 4：极端值驱动，谨慎解读。
3. **year 是"黄金特征"**：重要性最高（0.2690）、R² 最高（0.8595）、Ratio 最低（1.71）——四维都优秀。
4. **Diagnostic.means 的 Ratio=17.17**：极端的教学示例，展示了"密集区几乎平坦，稀疏区剧烈变化"的模式。
5. **高 R² ≠ 可信赖**：高 R² 可能来自极端值的杠杆效应。需要结合 Ratio 判断模式可信度。
6. **三维评估框架**：重要性 + 非线性（ΔR²）+ 交互强度 + 密度风险（Ratio），四维组合才能给出完整的特征诊断。
7. **实践影响**：
   - 临床解读：区分"全范围可信"和"边缘主导"特征。
   - 论文写作：在"局限性"部分讨论数据稀疏性。
   - 特征工程：Low Ratio 无需处理，Medium 做 Winsorization，High 检查极端值。
